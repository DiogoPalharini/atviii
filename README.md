# AutoManager - ATVIII

Sistema de gerenciamento de clientes e automóveis desenvolvido com Spring Boot.

## Pré-requisitos

- Java 17 ou superior
- Maven (ou use o Maven Wrapper incluído)

## Como Executar

### Windows:
```bash
cd automanager
.\mvnw.cmd spring-boot:run
```

### Linux/Mac:
```bash
cd automanager
./mvnw spring-boot:run
```
